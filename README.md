# 🌍 Mazen Emad Fawzy — Portfolio

A modern, visually stunning personal portfolio built with **React** and **Vite**, featuring a cinematic Earth-from-space hero section, smooth scroll-reveal animations, and a dark premium UI theme.

🔗 **Live Demo:** [maze-n01.github.io/my-portfolio](https://maze-n01.github.io/my-portfolio/)

---

## ✨ Features

- **Cinematic Hero** — Full-viewport Earth-from-space background with gradient overlays and animated text
- **Smooth Scroll Animations** — Intersection Observer-powered reveal effects with staggered timing
- **Responsive Design** — Fully responsive across desktop, tablet, and mobile
- **Interactive Radar Chart** — Skills visualization using Recharts
- **Dark Premium Theme** — Carefully crafted color palette with subtle glassmorphism effects
- **Starfield Background** — Animated canvas-based particle starfield
- **Accessible** — Respects `prefers-reduced-motion` for users who prefer no animations

## 🛠️ Tech Stack

| Technology | Purpose |
|-----------|---------|
| React 18 | UI Framework |
| Vite 6 | Build Tool & Dev Server |
| Recharts | Data Visualization (Radar Chart) |
| Lucide React | Icon Library |
| GitHub Pages | Hosting & Deployment |
| GitHub Actions | CI/CD Pipeline |

## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18+)
- npm

### Installation

```bash
# Clone the repository
git clone https://github.com/MAZE-N01/my-portfolio.git
cd my-portfolio

# Install dependencies
npm install

# Start development server
npm run dev
```

The site will be available at `http://localhost:5173/`

### Build for Production

```bash
npm run build
```
