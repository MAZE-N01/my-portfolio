# Mazen Fawzy — Portfolio

Personal portfolio site built with **React** and **Vite**. Dark theme, scroll-based reveal animations, and deployed via GitHub Pages.

🔗 **Live:** [maze-n01.github.io/my-portfolio](https://maze-n01.github.io/my-portfolio/)

---

## Overview

- Single-page layout with smooth section transitions
- Earth-from-space hero with gradient overlays
- Responsive across desktop and mobile
- Skills radar chart (Recharts)
- Animated starfield canvas background
- Respects `prefers-reduced-motion`

## Tech Stack

| Technology | Purpose |
|-----------|---------|
| React 18 | UI Framework |
| Vite 6 | Build Tool & Dev Server |
| Recharts | Data Visualization |
| Lucide React | Icons |
| GitHub Pages | Hosting |
| GitHub Actions | CI/CD |

## Setup

```bash
git clone https://github.com/MAZE-N01/my-portfolio.git
cd my-portfolio
npm install
npm run dev
```

Dev server runs at `http://localhost:5173/`

```bash
npm run build    # production bundle
```
